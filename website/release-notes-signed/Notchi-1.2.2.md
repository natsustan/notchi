# Notchi 1.2.2

This release adds Korean and Vietnamese localization, new panel behavior options, and a reorganized settings menu.

## Localization

Notchi now speaks Korean and Vietnamese.

- Localizes all user-facing text across the notch, usage pages, and settings
- Follows your system language automatically

## Panel

- Expands the panel when you hover over the notch and collapses it when you leave (optional, off by default)
- Adds a Hide Grass Island setting for a compact expanded panel

## Settings

- Reorganizes settings into General and Appearance pages so the main list stays short
- Keeps Agent Hooks, Usage, Emotion Analysis, and updates on the main settings page

## Fixes

- Shows the notch usage ring and session usage bar for Codex plans that only report a weekly limit
- Colors the session mode badge by the underlying permission mode in every locale
- Recognizes the free-text question option in question prompts regardless of language
