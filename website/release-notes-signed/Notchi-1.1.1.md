# Notchi 1.1.1

A small quality update with custom notification sounds and clearer settings.

## Custom Sounds

Notchi can now use your own notification sounds for completion and attention cues.

- Adds custom sound import support from the settings panel
- Lets you preview custom sounds without waiting for a live session event
- Keeps overlapping previews from cutting each other off

## Settings And Usage Polish

The expanded settings panel is easier to scan and clearer about the current agent state.

- Groups agent hook controls into a cleaner hooks section
- Moves the pin control into the settings header
- Shows Codex automatic state in the usage badge
- Softens setup colors and refreshes usage row wording

## Fixes

- Fixes usage bar fill so it stays clamped to its container
- Quiets noisy normal-path logs across services
