# Notchi 1.1.3

Notchi can now help answer Claude and Codex prompts directly from the notch, with smoother handling for interactive sessions.

## Inline Replies

Interactive agent prompts are easier to answer without leaving your flow.

- Answer Claude AskUserQuestion prompts from the notch
- Submit grouped question answers as one complete response
- Reply to free-text options inline
- Use number keys or click to pick an option
- Cancel pending questions with Escape

## Permission Prompts

Notchi now surfaces more actionable agent requests where you can respond quickly.

- Answer Claude permission requests from the notch
- Show Codex prompts as display-only when Codex does not expose a reply action
- Wiggle the reply hint when tapping display-only Codex options

## Fixes

- Cleans up text field state after submitting with Return
