# Notchi 1.2.1

This release adds Japanese and Chinese localization, richer cost-chart details, and quicker access to usage from the notch.

## Localization

Notchi now speaks Japanese and Simplified / Traditional Chinese.

- Localizes all user-facing text across the notch, usage pages, and settings
- Formats counts and dates using your locale's conventions

## Usage & Cost

- Shows the hovered day's top model on the cost chart
- Opens the usage page when you click the collapsed notch ring
- Opens the usage page on your last-used provider when idle
- Keeps the last-viewed session on the collapsed notch until newer activity arrives
- Supports a custom API base URL for emotion analysis providers

## Performance

- Scans Codex usage transcripts incrementally instead of re-reading them
- Coalesces compaction refreshes and caches Codex session discovery
- Caches sprite sheet lookups and honors Reduce Motion for grass sprites

## Fixes

- Labels Codex rate-limit windows by their actual size instead of slot order
- Keeps cost stat values from resizing on hover and shortens long GPT model names
- Shows zero tokens in the today stat when today has no usage
- Keeps pricing refresh working when models.dev changes its data shape
- Aligns the collapsed ring's click target with the displayed sprite
